Style guide

It's dangerous to go alone…

This Style & Pattern library is a collection of Atomic elements that come together to build Cards Against.

Using this technique we can build ideas quickly and easily, keeping everything all uniform and lovely.

This library is here to help you, it's a bit like the Twitter bootstrap — but it's only for this site so it's not as big.

Whether you're working on a new feature, modifying an existing one or whatever it is you're doing — use this library. If we start deviating from this library we start getting inconsistencies in our app and we make it harder to update further down the line.

If you want to add a new feature and the patterns/styles don't exist in this library, check again. If you're still 100% sure they don't exist then create a new pattern/style and add it in to the library. Do NOT just write it into the app — It's lazy.

Nested grids

You can also nest grids, but be careful. This can go very wrong, very quickly. If you want to group them just add a class of .group and it will remove the extra padding.

Stick to a maximum of three nested elements – DON'T go any further than this.

Typography

The typography on SuperCleverStuff follows the typeplate stucture with a 0.16803em baseline. It's limited to a 38em line-width by default which keeps it under the suggested 72 character line-length, makes the text easier to read.
